# Memory Flash

## environment setup

### Python

*dependencies*

	pip3 install sanic
	pip3 install jinja2
	
*run*

	python3 main.py
	
*test*
	
	python3 -m unittest discover

### Javascript

*dependencies*

	npm install mocha --save-dev

*test*


	npm test
