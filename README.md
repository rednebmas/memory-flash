### Python

*install*

	pip3 install -r requirements.txt
	
*run*

	python3 main.py
	
*test*
	
	python3 -m unittest 

### Javascript

*dependencies*

	npm install 
	sudo gem install sass

*test*

	npm test

*dev in browser*

	npm run hot-compile
